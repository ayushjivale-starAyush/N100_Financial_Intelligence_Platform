# N100 Financial Intelligence Platform

## Sprint 1 – Data Foundation

### Objective

Build and validate a SQLite database named `nifty100.db` using financial datasets and ETL pipelines.

### Technologies Used

- Python
- Pandas
- NumPy
- SQLite
- SQLAlchemy
- Pytest
- Jupyter Notebook

### Folder Structure

- data/raw : Raw source files
- data/processed : Cleaned files
- src/etl : ETL scripts
- tests/etl : Unit tests
- db : Database and schema
- output : Validation reports
- notebooks : SQL queries
